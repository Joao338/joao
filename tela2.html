Afinal, o que é Html:

    HTML, OU HyperText Markup Language (linguagem de Marcação de Hipertexto), é uma liguagem de Hipertexto), é uma liguagem de maracação usada para criar e estruturar conteudo na we. Foi desenvolvida comoa linguagem padrão para criar páginas da web,. Foi desenvolvido como a linguagem padrão para criar paginas da web e é usado para definir a estrutura, o layout e o significado do contuedo em documento. HTML permite que você organize e apresente informações, imagens, links e outros elementos multimídia de forma coerente e consistente em navegadores da web
